# Odin Restaurant Page

A small single-page restaurant website built as part of The Odin Project's "Restaurant Page" assignment.

Assignment: https://www.theodinproject.com/lessons/node-path-javascript-restaurant-page

## Description

This project implements a simple restaurant website using vanilla JavaScript, HTML, and CSS with a modular file structure. The site includes a home page, menu page, and contact page and demonstrates DOM manipulation and dynamic content switching without reloading the page.

## Features

- Multi-page view (home, menu, contact) handled via JavaScript modules
- Responsive layout and basic styling
- Image assets included in the `public`/`src/images` folders

## Tech Stack

- HTML
- CSS
- JavaScript (ES6 modules)
- Webpack (for bundling)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rimanz/odin-restaurant.git
   cd odin-restaurant
   ```
2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Build for production:

   ```bash
   npm run build
   ```

## Project Structure

- `src/` - source files (JavaScript modules, styles, templates)
- `public/` - static assets and images
- `webpack.config.js` - build configuration
- `package.json` - project metadata and scripts

## Usage

Open the development server URL shown by your bundler (commonly `http://localhost:8080`), or open the generated `dist/index.html` after running the build.
